# ⚽ Статистика ЧМ-2026 · Табло

Streamlit-приложение со статистикой Чемпионата мира по футболу 2026. Данные
тянутся **живьём** через Anthropic API с инструментом `web_search`.

Три раздела:

1. **Бомбардиры — Топ-5** (имя, сборная, голы). Источник: FIFA.
2. **Ассистенты — Топ-5** (имя, сборная, ассисты). Источник: FIFA.
3. **Стоимость сборных — Топ-20** (€ млн, по убыванию, с полосами-барами).
   Источник: Transfermarkt.

У каждого раздела есть кнопка **«Обновить»**, плюс общая кнопка **«Обновить
всё»**. Перед названием сборной — флаг-эмодзи. Тёмная тема в стиле табло:
тёмный фон, зелёный акцент, золото для стоимости. Результаты кэшируются
(`st.cache_data`, ttl 600 сек), чтобы не дёргать API лишний раз.

> ⏱ Турнир стартовал 11 июня 2026. На раннем этапе списки бомбардиров и
> ассистентов могут быть короткими или пустыми — это нормально.

---

## 🚀 Запуск локально

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Прописать ключ Anthropic
#    Откройте .streamlit/secrets.toml и вставьте свой ключ:
#    ANTHROPIC_API_KEY = "sk-ant-..."

# 3. Запустить
streamlit run app.py
```

Приложение откроется на http://localhost:8501.

> Файл `.streamlit/secrets.toml` добавлен в `.gitignore` — ключ **не** попадёт
> в репозиторий. В репозитории лежит только плейсхолдер.

---

## ☁️ Деплой на Streamlit Community Cloud

1. Зайдите на **https://share.streamlit.io** и войдите через GitHub.
2. Нажмите **New app**.
3. Выберите:
   - **Repository:** `Azeeemaaa/wc2026-stats`
   - **Branch:** `main`
   - **Main file path:** `app.py`
4. Откройте **Advanced settings… → Secrets** (или после деплоя
   **Settings → Secrets**) и вставьте:

   ```toml
   ANTHROPIC_API_KEY = "sk-ant-ваш-настоящий-ключ"
   ```

5. Нажмите **Deploy**. Через минуту приложение будет доступно по публичной
   ссылке вида `https://<app-name>.streamlit.app`.

> Ключ Anthropic берётся из `st.secrets["ANTHROPIC_API_KEY"]` и нигде не
> хардкодится в коде.

## 🧩 Как устроены данные

Для каждой категории отправляется отдельный запрос к модели
`claude-sonnet-4-6` (`max_tokens=1000`) с инструментом
`web_search_20250305`. Промт просит вернуть **только** JSON в строгой схеме;
из ответа берутся текстовые блоки, убираются ```-обёртки, парсится подстрока
от первой `{` до последней `}`.

Раздел «Стоимость сборных» при первом запуске заполнен стартовым снимком
(хардкод), кнопка «Обновить» перезаписывает его живыми данными.
